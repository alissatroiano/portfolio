AI Art & Image Generation

Hue

Hue is a digital art store and image generation app with custom machine learning models powered by MindsDB and OpenAI's Dall-E model. Hue's backend is powered by Python and Django and the shopping cart checkout functionality uses the Stripe API.

Python, Django, MindsDB, OpenAI, JavaScript, HTML, CSS, Bootstrap 5, AWS S3, Heroku, Stripe API, Git
